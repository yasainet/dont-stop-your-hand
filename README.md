A Chrome extension that prevents videos from pausing when you switch tabs on MissAV.

## Features

- Disables auto-pause when switching tabs
- Enables background playback
- Disables Plyr player's autopause feature

## Supported Sites

- missav.ws
- missav.live
- missav.ai
- njavtv.com

## Installation

This extension is not available on the Chrome Web Store (developer registration requires payment).
Please install manually using the steps below.

### 1. Download

**Option A: Download ZIP**

1. Click the green "Code" button at the top of this page
2. Select "Download ZIP"
3. Extract the downloaded ZIP file

**Option B: git clone**

```bash
git clone https://github.com/yasainet/dont-stop-your-hand.git
```

### 2. Load into Chrome

1. Open `chrome://extensions` in Chrome
2. Enable "Developer mode" in the top right corner
3. Click "Load unpacked"
4. Select the downloaded folder

### 3. Done

The extension will automatically work when you visit supported sites.

## Uninstall

1. Open `chrome://extensions`
2. Click "Remove" on "Don't Stop Your Hand"
